链接

链接路径

绝对路径：

  • 指定从根目录到文件的完整路径。

示例：

<a href="D:\文件\文件\html.html"></a>

相对路径

  • 指定相对于当前文件的文件位置。

示例：

<a href="html.html"></a>

<a> 标签

  • <a> 标签定义超链接，用于从一个页面链接到另一个页面。
  • <a> 元素最重要的属性是 href 属性，它指定链接的目标。

示例：

<a href="地址">文本</a>

在所有浏览器中，连接的默认外观如下

  • 未被访问的链接带有下划线而且是蓝色的
  • 已被访问的链接带有下划线而且是紫色的
  • 活动链接带有下划线而且是红色的

锚点

示例：

<a href="#pantu">跳转到叛徒</a>
<p>我们中出了个叛徒</p>
<p>我们中出了个叛徒</p>
<p><a name="pantu">我是叛徒</a></p>
<p>我们中出了个叛徒</p>

效果：

跳转到叛徒

我们中出了个叛徒

我们中出了个叛徒

我是叛徒

我们中出了个叛徒

点击后，你会发现页面最顶端跳转到的锚点的位置
