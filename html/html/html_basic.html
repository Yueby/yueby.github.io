HTML文档基本结构

示例：

<!doctype html>
<html>

 <head>
  <title>我的第一个网页</title>
 </head>

 <body>

 </body>
</html>
  • <!doctype html> 声明为 HTML5 文档，并且<!doctype>声明必须位于 HTML5 文档中的第一行,使用非常简单
  • <html>元素是 HTML 页面的根元素
  • <head> 元素包含了文档的元（meta）数据，如<meta charset="utf-8"/>定义网页编码格式为 utf-8。
  • <meta charset="utf-8">定义网页编码格式为 utf-8。
  • <title>元素描述了文档的标题
  • <body>元素包含了可见的页面内容
