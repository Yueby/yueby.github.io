<meta>标签

网页页面中出现乱码，那如何消除乱码使其正常显示？
我们可以使用<meta>标签来解决。

示例：

<head>
 <meta charset="utf-8" />
 <title>网页标题</title>
</head>

  • HTML 4.01： <meta http-equiv="content-type" content="text/html; charset=UTF-8">
  • HTML5： <meta charset="UTF-8">
  • charset属性可以设置各种编码
