表格和边框属性

border属性

如果不定义边框属性，表格将不显示边框。有时这很有用，但是大多数时候，我们希望显示边框。

使用边框属性来显示一个带有边框的表格：

示例：

<table border="1">
 <tr>
  <td>Row 1, cell 1</td>
  <td>Row 1, cell 2</td>
 </tr>
<table>

效果：

Row 1, cell 1 Row 1, cell 2

想要取消表格边框只需要将 border 属性的值改为 0 就可以了。

示例：

<table border="0">
 <tr>
  <td>Row 1, cell 1</td>
  <td>Row 1, cell 2</td>
 </tr>
<table>

效果：

Row 1, cell 1 Row 1, cell 2

cellspacing属性

cellspacing 属性规定单元之间的空间，以像素计。若不设置该属性，则其默认值为 cellspacing="2"。

以下示例取消了表格单元之间的空间。

示例：

<table border="1" cellspacing="0">
 <tr>
  <td>Row 1, cell 1</td>
  <td>Row 1, cell 2</td>
 </tr>
<table>

效果：

Row 1, cell 1 Row 1, cell 2

cellpadding属性

cellpadding 属性规定单元边沿与单元内容之间的空间，以像素计。

以下示例是 没有单元格边距的表格 和 有单元格边距 的表格。

示例：

<table border="1">
 <tr>
  <th>Month</th>
  <th>Savings</th>
 </tr>
 <tr>
  <td>January</td>
  <td>$100</td>
 </tr>
</table>
<table border="1" cellpadding="10">
 <tr>
  <th>Month</th>
  <th>Savings</th>
 </tr>
 <tr>
  <td>January</td>
  <td>$100</td>
 </tr>
</table>

效果：

Month Savings
January $100
Month Savings
January $100

表格的跨行与跨列

colspan 属性 与 rowspan 属性

colspan 和 rowspan 属性可以使用在table内标签上，如 <th>、<tr>、<td> 标签上。

示例：

<table border="1">
 <tr>
  <th rowspan="2">跨两行</th>
  <th>row</th>
 </tr>
 <tr>
  <td>row</td>
 </tr>
</table>
<table border="1" cellpadding="10">
 <tr>
  <th colspan="2">跨两列</th>
 </tr>
 <tr>
  <td>col</td>
  <td>col</td>
 </tr>
</table>

效果：

跨两行 row
row
跨两列
col col
